# Obsidian Web Clipper Templates

## Usage
1) Install [Obsidian Web Clipper](https://help.obsidian.md/web-clipper)
2) [Import my templates into Web Clipper](https://help.obsidian.md/web-clipper/templates#Import+and+export+Web+Clipper+templates)

## Websites
1) `apple-music.json` - Add album, EP or single from Apple Music
- Replace mentions of 500x500 in template to desired resolution you want your cover art to be fetched in. (Max = 5000x5000)
- Note: large resolutions will make Obsidian lag. So after importing with the template, run the command "Download attachments for current file" to save images locally.

2) `musicbrainz-release.json` - Add any release from MusicBrainz

## Examples
Example markdown files are located in the `examples` directory. No promises on keeping them updated, so the best way to test the templates is to try them yourselves :)

## Blacklisted Websites
1) [albumoftheyear](albumoftheyear.org) - utilise javascript for most of the content
2) [Spotify](open.spotify.com) - same as aoty
3) [YouTube Music](music.youtube.com) - same as aoty
